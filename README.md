Number Pyramid Pattern in Java

This program prints a number pyramid pattern.

First, we take a variable n which decides the number of rows.

The outer loop runs from 1 to n to print each row.

The first inner loop prints spaces so the pyramid stays centered.

The second loop prints numbers from 1 to i in increasing order.

The third loop prints numbers from i-1 to 1 in decreasing order.

Together these loops create a pyramid pattern using numbers.
